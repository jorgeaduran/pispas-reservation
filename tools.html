ERRORES

⚡ Manejo de Errores

SNAFU
Errores personalizados con contexto automático
#[derive(Debug, Snafu)]
enum Error {
  #[snafu(display("User {} not found", id))]
  UserNotFound { id: u32 },
}

ensure!(user.exists(), UserNotFoundSnafu { id });
🎯 anyhow
Manejo simple de errores para aplicaciones
